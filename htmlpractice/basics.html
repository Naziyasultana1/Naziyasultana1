HTML has elements which has start tag and end tag firstly, html tag is the root of the code and there is a head tag .ommonly head tag has five there are title tag to write a title ,meta tag for rules and regulations of the webpage. script tag is used to add javascript style tag is used to as css styles and link tag which is used to add links in the webpage .
